# 理论
考核方式：**机考**（电脑上做题）  
合格分数：60分  
题目：100道单选，40道多选，60道判断（0.5分/题，共100分）  
如何练习：
1. 购买题库软件：如`消考宝`，为防止一些药友下错，我这里提供一下网址：[消考宝官网](https://www.119xkb.com/ "消考宝官网")  
花费：43.78元（当前价格时间:2026.04.22）

2. 直接从咸鱼等渠道购买理论题库,搜索`消控证理论题库`  
	花费：20元以下  
	不用担心题库更新不及时，从我24年考初级到现在考中级，中间的题目几乎没有什么变化，只要平时练习得足够好，肯定能过  
	
3. 如果妖友有其他的方式可以在下面评论出来，大家一起学习  

![考试成绩](https://i1.ipix.ink/m/36/2026/04/69e77e097a8ee.webp)



# 实操
考核方式：面考（考官询问，你口述操作过程或实际操作）  
总分：100分  
合格分数：60分  
题目：必考+抽考  


>[!NOTE]
> 必考题：属于必考题的题目考试时一定会考到  
> 抽考题：属于抽考题的题目考试时从中抽取几道考试

如何练习：
1. 教安VR，官网网址：[教安vr](https://www.bjzjxf.com/001 "教安vr")  
3D全景实操模拟：199元，实操精讲视频：199元（当前价格时间:2026.04.22）  
想省一点可以只买`3D全景实操模拟`，购买的时候注意`考区选择`你报名的鉴定站，不然每个鉴定考的有区别，到时候考试一脸懵  

2. 不买`实操精讲视频`  
a)先通过b站、抖音等视频平台搜索你报名的鉴定站，网上应该有流传出响应考试设备的型号和现场设备的放置图，比如我这里就是海湾5000/5000H和松江9108这类主机  
b)通过搜索相应设备，b站、抖音等视频平台直接搜索`设备名称`+`实操教程`，然后系统性的观看  
c）可以在咸鱼找一点视频资料，应该花费20元以下，再将b的内容查漏补缺  

3. 其他练习方式

# 备考时间安排
时间最好还是把握一下，像我时间安排就有点紧了，理论学了3天，实操学了4天，时间再长一点我考试的时候能更有把握  
理论：咸鱼购买题库的话，从购买那一刻就可以开始看了，不然时间有点紧  
实操：可以先通过咸鱼等平台购买的实操视频

# 其他疑问
1.背锅证？
>这个我确实不知道，我不是做这个工作的，我是做安全的，只是对这个岗位有一部分了解，不做评价，具体的可以看一看做相关工作的妖友的帖子

2.要入刑？
>a）紧急来时未及时响应，如睡着了，玩手机没注意到等，或者报警不及时、设备未启动等导致严重后果的都会入刑，基本上做到了紧急情况是设备打到自动、报告警情及时就不会入刑，追究不到你这里，可以多看一看事故调查报告  
>b）被领导安排杂活时导致未能响应，这种情况你和领导都可能入刑，看具体的事故调查，如果你有领导安排你去干杂活的电话录音（安卓手机都有内置的电话、微信电话等自动录音功能，自己搜索教程打开）、聊天记录等，你的责任就能小一点，最好有明确拒绝的记录，然后是被领导强令要求去做的记录  
>c)把握住这三者违规行为、严重后果、因果关系
